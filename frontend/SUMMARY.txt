API INTEGRATION
================================================================================

Authentication Endpoints:
  POST   /auth/login - Login with email/password
  POST   /auth/register - Register new user
  GET    /auth/me - Get current user
  POST   /auth/refresh - Refresh JWT token
  POST   /auth/logout - Logout user

Release Endpoints:
  GET    /releases - List releases with pagination
  GET    /releases/{id} - Get single release
  POST   /releases - Create new release
  POST   /releases/{id}/deploy - Deploy to environment
  GET    /releases/history - Get release history

Deployment Endpoints:
  GET    /deployments - List deployments
  GET    /deployments/{id} - Get single deployment
  POST   /deployments/{id}/approve - Approve deployment
  POST   /deployments/{id}/reject - Reject deployment
  POST   /deployments/{id}/rollback - Rollback deployment
  GET    /deployments/{id}/stages - Get pipeline stages
  GET    /deployments/{id}/logs - Get deployment logs

Analytics Endpoints:
  GET    /analytics/metrics/summary - DORA metrics summary
  GET    /analytics/metrics/trends - Deployment trends
  GET    /analytics/metrics/by-service - Service metrics
  GET    /analytics/metrics/by-environment - Environment metrics
  GET    /audit-logs - Audit log entries

All endpoints support:
  • Pagination (page, page_size parameters)
  • Filtering (status, environment, service_id, etc.)
  • Sorting
  • Date range filtering
  • Error handling with proper HTTP status codes
